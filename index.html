Apa itu AMP?

Accelerated Mobile Pages (AMP) adalah sebuah proyek open-source dari Google yang dirancang untuk membuat halaman web yang dimuat dengan sangat cepat di perangkat mobile. AMP mengoptimalkan kinerja dan pengalaman pengguna dengan membatasi penggunaan JavaScript dan CSS yang tidak perlu.

AMP menggunakan arsitektur yang dikurasi dengan ketat, runtime JavaScript bersyarat, dan caching yang dikelola untuk memastikan kinerja yang konsisten dan cepat.

Keuntungan AMP

  • Kecepatan Loading Ekstrem - Halaman dimuat dalam hitungan milidetik
  • Pengalaman Mobile Pertama - Dioptimalkan untuk perangkat mobile
  • Konsumsi Data Rendah - Lebih efisien dalam penggunaan bandwidth
  • Better SEO - Google memberikan prioritas untuk halaman AMP
  • Engagement Tinggi - User experience yang lebih baik meningkatkan engagement
  • Universalitas - Berfungsi di semua browser modern

Komponen Dasar AMP

Setiap halaman AMP harus memiliki:

  • Doctype AMP - <!doctype html> dengan atribut ⚡
  • Meta Charset - UTF-8 charset declaration
  • Viewport Meta Tag - Responsive design settings
  • Canonical Link - Link ke versi original
  • AMP Runtime - Script dari cdn.ampproject.org
  • AMP Boilerplate CSS - CSS wajib untuk AMP

Halaman ini adalah contoh halaman AMP yang valid dan telah mengikuti semua persyaratan AMP.
